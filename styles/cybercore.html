Three rules of the console

$ man cybercore — the protocol behind the phosphor.

RULE0x01
01

monospace only

Every glyph occupies one cell. Fixed-width type is the grid the whole interface snaps to — code, labels and readouts alike.

RULE0x02
02

light is data

Glow is never decoration. Phosphor bloom marks active state; scanlines and flicker simulate the CRT carrying a live signal.

RULE0x03
03

frame everything

Each panel is a terminal window — title bar, corner ticks, process ID. Structure reads as a stack of running programs.
